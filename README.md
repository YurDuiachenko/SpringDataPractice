# SpringDataPractice
### Описание
REST API сервис:
- 4-5 сущностей со связями many-to-many и one-to-many
- CRUD-репозитории
- Кастомные запросы под сценарии использования
- REST-контроллеры + DTO Automapping

[_Java+SpringDataJPA+PostgreSQL+Maven+Postman_]
